# 工厂实拍图库

> AI 自动配图系统的**第一优先级图源**。当文章配图时，会**优先**从本目录挑选工厂真实场景图。

## 📁 文件命名规范

```
<场景代号>-<序号>.<扩展名>
```

| 代号 | 场景 |
|---|---|
| `machine-` | 数码印花机、生产线 |
| `fabric-` | 面料特写、半成品 |
| `product-` | 成品展示（窗帘/墙布/家纺） |
| `workshop-` | 车间全景、工人操作 |
| `qc-` | 品控、检验、验厂 |
| `client-` | 客户来访、合作场景 |
| `sample-` | 打样、花型、配色 |

**示例**：
- `machine-01.jpg` — 第一台数码印花机
- `fabric-03.png` — 第三张面料特写
- `product-02.jpg` — 第二款成品（窗帘）

## 🎯 AI 关键词匹配规则

文章配图时按以下顺序**优先匹配**（一个图可被多关键词命中）：

1. **标题含"采购/询价/价格"** → 优先 `client-`、`workshop-`、`qc-`
2. **标题含"工艺/技术/印花"** → 优先 `machine-`、`fabric-`
3. **标题含"案例/应用/成品"** → 优先 `product-`、`sample-`
4. **标题含"品质/环保/OEKO"** → 优先 `qc-`、`fabric-`
5. **兜底** → 按文件名字母顺序轮询

## 📐 图片要求

- 比例：**16:9 或 4:3**（适配文章展示）
- 分辨率：**≥ 1200×800**（保证 Retina 清晰）
- 大小：**≤ 500KB**（GitHub 仓体积友好）
- 格式：`.jpg` / `.png` / `.webp`
- 命名：**英文 + 数字 + 扩展名**（无中文、空格、特殊字符）

## 🔄 同步到文章

AI 自动配图流程：
```
文章配图请求
   ↓
1. 查 factory-photos/ 关键词匹配 → 命中即用
   ↓ 未命中
2. 调 Pexels API（兜底）
   ↓ 失败
3. 生成 SVG 占位图（绝不卡死）
```

## ➕ 添加新图步骤

1. 拍照后压缩到 ≤ 500KB（用 [tinypng.com](https://tinypng.com)）
2. 命名按上述规范
3. 拖到本目录
4. Commit + push（老板手动 git push，或通过 GH Actions 自动同步）
5. **明天 02:00** GH Actions 会**自动用上**新图

> 💡 **建议起步**：先放 8-10 张机器+面料+成品的实拍图，文章配图质量会立刻大幅提升。
